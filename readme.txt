Projekt "Analizator tekstu" przeznaczony jest do wyszukiwania we wprowadzonym tekście elementów pasujacych to wskazanego wzorca.
Wzorcem moze być:
- konkretna fraza,
- wyraz o określonej długości,
- liczby o określonej ilosci cyfr,
- numery telefonów, cytaty, kody pocztowe oraz adresy email.

Dla wyników wyszukiwania można ustalić podstawowe stylowanie takie jak kolor, pogrubienie i podkreślenie.

Za wyszukiwanie w tekście odpowiadają wyrażenie regularne i metoda search.

Użyte technologie: HTML, CSS (+SCSS) i JavaScript.
